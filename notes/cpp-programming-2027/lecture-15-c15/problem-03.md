# 문제 3 · 변수의 주소와 첫 바이트 (O/X)

::: info 문제 3 [1점]
아래 문장을 읽고 O/X로 답하세요.

> C++ 변수의 주소<span class="gloss">address</span>는 해당 변수에 할당된 첫째 바이트의 주소이다.
:::

::: details 풀이 및 해설
**정답: O**

변수는 자료형의 크기만큼 연속된 여러 바이트를 차지합니다(예: `int`는 보통 4바이트). 주소 연산자 `&`가 돌려주는 그 변수의 주소는 **변수가 차지하는 여러 바이트 중 첫째(가장 낮은 번지) 바이트의 주소**입니다.

예를 들어 어떤 `int` 변수가 `00FF00`\~`00FF03`의 4바이트를 차지한다면, `&num`은 시작 번지인 `00FF00`을 가리킵니다. 배열도 마찬가지로 배열의 주소는 첫 원소의 첫 바이트 주소와 같습니다.

따라서 문장은 참입니다.

$$\boxed{\text{O}}$$
:::
