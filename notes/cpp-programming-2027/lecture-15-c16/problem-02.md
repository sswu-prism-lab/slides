# 문제 2 · 포인터 식별자 `p` 접두 관습

::: info 문제 2 [1점]
아래 문제를 읽고 O/X로 답하세요.

C++ 코딩 시, 관습적으로 포인터의 식별자는 참조하고자 하는 변수 이름 앞에 `p`를 붙인다. (아래는 예시)

```cpp
double density = 0.5;
double* pDensity = &density;
```
:::

::: details 풀이 및 해설
**답: `O`**

포인터 변수임을 이름만 보고도 알 수 있도록, 가리키는 대상 변수 이름 앞에 소문자 `p`를 붙이는 것이 널리 쓰이는 **명명 관습<span class="gloss">naming convention</span>**입니다. 예시의 `density`(대상)에 대응하는 포인터를 `pDensity`로 지은 것이 그 예입니다.

이는 문법이 강제하는 규칙은 아니지만, 코드 가독성을 높이기 위한 관습으로 통용됩니다.

$$\boxed{\text{O (참)}}$$
:::
