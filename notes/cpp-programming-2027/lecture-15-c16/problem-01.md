# 문제 1 · 기본 소멸자의 자동 생성

::: info 문제 1 [1점]
아래 문제를 읽고 O/X로 답하세요.

C++에서 소멸자<span class="gloss">destructor</span>가 명시적으로 정의되지 않은 경우, 모든 클래스는 기본 소멸자를 포함한다.
:::

::: details 풀이 및 해설
**답: `O`**

프로그래머가 소멸자를 직접 작성하지 않으면, 컴파일러가 **암묵적으로 기본 소멸자<span class="gloss">default destructor</span>를 자동 생성**합니다. 이 기본 소멸자는 몸체가 비어 있으며(`~ClassName() {}` 형태), 멤버들의 소멸자를 역순으로 호출해 줍니다.

즉 소멸자를 명시하지 않은 클래스라 하더라도, 객체가 수명을 다할 때 호출될 소멸자는 항상 존재합니다.

$$\boxed{\text{O (참)}}$$
:::
