# 문제 5 · `static_cast`와 문자 산술

::: info 문제 5 [2점]
아래 코드를 읽고, 출력을 예상하여 작성하세요.

```cpp
double x = static_cast<double>(3 / 6);
double y = x *= 2;
cout << static_cast<char>('a' + ++y) << '\"'
    << static_cast<char>('a' + x++) << '\"';
```
:::

::: details 풀이 및 해설
**변수의 초기값부터** 정확히 잡는 것이 관건입니다.

- `3 / 6`은 **정수 나눗셈**이라 `0`. 그 뒤 `static_cast<double>`을 해도 이미 `0`이므로 `x = 0.0`.
  (형변환은 나눗셈이 끝난 다음에 적용됨에 주의)
- `x *= 2`는 `x = 0.0 * 2 = 0.0`이고, 그 대입식의 값이 `y`에 들어가 `y = 0.0`.

이제 출력식을 왼쪽에서 오른쪽으로 계산합니다.

1. `++y`는 **전위 증가**<span class="gloss">pre-increment</span>라 `y`를 먼저 `1.0`으로 만든 뒤 그 값을 사용 → `'a' + 1.0 = 97 + 1 = 98` → `static_cast<char>(98)` = **`b`**
2. `'\"'` → 큰따옴표 문자 `"`
3. `x++`는 **후위 증가**라 현재 값 `0.0`을 먼저 사용하고 `x`는 `1.0`이 됨 → `'a' + 0.0 = 97` → `static_cast<char>(97)` = **`a`**
4. `'\"'` → 큰따옴표 문자 `"`

이어 붙이면 출력은 `b"a"` 입니다. (줄바꿈 없음)

$$\boxed{\texttt{b"a"}}$$

:::
