# 문제 8 · 직각삼각형 별 출력

::: info 문제 8 [2점]
아래 문제를 풀기 위한 의사알고리즘<span class="gloss">pseudo-algorithm</span>을 작성하세요.

임의의 정수를 입력받아 아래처럼 `*`로 이루어진 직각삼각형을 출력 (`setw(n)` 함수 사용 금지).
입력받은 정수가 4인 경우의 예:

```
*
**
***
****
```
:::

::: details 풀이 및 해설
`i`번째 줄에 별을 정확히 `i`개 찍으면 됩니다. 바깥 반복문이 줄(행) `i = 1..n`을, 안쪽 반복문이 그 줄의 별 개수 `j = 1..i`를 담당하는 **이중 반복문**으로 해결합니다.

**의사알고리즘**

```
1. 정수 n을 입력받는다.
2. i를 1부터 n까지 1씩 증가시키며 반복:
     a. j를 1부터 i까지 1씩 증가시키며 '*'를 (줄바꿈 없이) 출력한다.
     b. 한 줄을 다 찍었으면 줄바꿈을 출력한다.
```

`n = 4`이면 각 줄에 별이 `1, 2, 3, 4`개씩 찍혀 위 예시와 같은 직각삼각형이 됩니다. `setw`나 공백 정렬 없이 별 개수만으로 모양을 만듭니다.

:::
