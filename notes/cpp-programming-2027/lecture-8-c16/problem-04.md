# 문제 4 · 삼항 연산자와 후위 증가

::: info 문제 4 [1점]
아래 코드를 읽고, 출력을 예상하여 작성하세요.

```cpp
int a = 3, b = 3;
int c = a++ == b ? a++ : a--;
cout << c << endl;
```
:::

::: details 풀이 및 해설
후위 증가<span class="gloss">post-increment</span> `a++`는 **현재 값을 먼저 내놓은 뒤** `a`를 1 증가시킵니다.

1. 조건식 `a++ == b`: `a++`는 현재 값 `3`을 내놓고 `a`는 `4`가 됨 → `3 == 3` → **참**
2. 참이므로 삼항 연산자는 `a++` 가지를 계산: `a++`는 현재 값 `4`를 내놓고 `a`는 `5`가 됨
3. 그 결과 `4`가 `c`에 대입됨

따라서 `c`는 `4`이고, (참고로 `a`는 최종적으로 `5`) 출력은 `4` 입니다. 거짓 가지인 `a--`는 실행되지 않습니다.

$$\boxed{c = 4}$$

:::
