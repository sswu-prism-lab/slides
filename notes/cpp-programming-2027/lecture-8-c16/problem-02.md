# 문제 2 · 상수 식별자 명명 관습

::: info 문제 2 [1점]
아래 문장을 읽고 O/X로 답하세요.

C++ 코딩 시, 관습적으로 상수형<span class="gloss">constant</span>의 식별자는 대문자로만 구성한다.
예: `const double PI = 3.14159;`
:::

::: details 풀이 및 해설
답은 **O** 입니다.

이는 문법 규칙이 아니라 **코딩 관습**<span class="gloss">convention</span>에 대한 문제입니다. C++을 비롯한 많은 언어에서 컴파일 시점에 값이 고정되는 상수는 **모두 대문자**로(여러 단어면 `MAX_SIZE`처럼 밑줄로 구분) 적어, 일반 변수와 한눈에 구분되도록 하는 것이 널리 쓰이는 관습입니다.

```cpp
const double PI = 3.14159;      // 관습에 맞는 상수 이름
const int MAX_COUNT = 100;      // 여러 단어는 밑줄(_)로 연결
int studentCount = 0;           // 일반 변수는 소문자 카멜케이스 등
```

문법적으로는 소문자 상수도 허용되지만, 문제는 "관습적으로"라고 명시했으므로 서술은 옳습니다.

$$\boxed{\text{O}}$$
:::
