# 문제 3 · 정수 나눗셈과 연산자 우선순위

::: info 문제 3 [1점]
아래 코드를 읽고, 출력을 예상하여 작성하세요.

```cpp
cout << 2 + (4 / 6) * 4 - 2 << endl;
```
:::

::: details 풀이 및 해설
핵심은 **정수 나눗셈**입니다. `4`와 `6`은 모두 정수이므로 `4 / 6`은 실수 `0.666...`이 아니라 소수부를 버린 **정수 `0`** 이 됩니다.

연산 순서를 따라가면

1. `4 / 6` → `0`
2. `0 * 4` → `0` (곱셈이 덧셈·뺄셈보다 우선)
3. `2 + 0 - 2` → `0` (덧셈·뺄셈은 왼쪽에서 오른쪽)

$$2 + (4/6)\times 4 - 2 = 2 + 0 \times 4 - 2 = 0$$

따라서 출력은 `0` 입니다.

$$\boxed{0}$$
