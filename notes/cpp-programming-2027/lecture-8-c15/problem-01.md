# 문제 1 · 조건 연산자는 유일한 삼항 연산자인가

::: info 문제 1 [1점]
아래 문장이 옳으면 O, 틀리면 X로 답하세요.

C++에서 조건 표현식<span class="gloss">conditional expression</span>은 유일한 3항 연산자<span class="gloss">ternary operator</span>이다.
:::

::: details 풀이 및 해설
**답: O**

C++에는 피연산자를 세 개 받는 연산자가 조건 연산자 `?:` 하나뿐입니다.

```cpp
결과 = 조건 ? 참일_때_값 : 거짓일_때_값;
```

`조건`, `참일 때 값`, `거짓 때 값`의 **세 피연산자**를 취하므로 삼항(3항) 연산자이며, C++ 표준에 정의된 삼항 연산자는 이것 하나뿐입니다. 따라서 문장은 옳습니다.

$$\boxed{\text{O}}$$
:::
