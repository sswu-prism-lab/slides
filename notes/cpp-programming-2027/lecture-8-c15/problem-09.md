# 문제 9 · `switch`와 삼항 연산자

::: info 문제 9 [2점]
아래 코드의 `x`, `y`, `z`는 어떤 값을 갖는지 쓰시오.

```cpp
int x = 2, y = -1, z;
switch (--x + ++y) {
    case 0: z = x == 1 ? 1 : -1; break;
    case 1: z = (x * y) <= 0 ? 1 : -1; break;
    default: z = y > -1 ? 1 : -1;
}
```
:::

::: details 풀이 및 해설
**1) `switch` 조건 계산**
- `--x`: `x`가 `1`이 되고 그 값 `1`을 사용
- `++y`: `y`가 `0`이 되고 그 값 `0`을 사용
- 합 `= 1 + 0 = 1` → **`case 1`** 실행

이 시점에서 `x = 1`, `y = 0`입니다.

**2) `case 1` 실행**
- `z = (x * y) <= 0 ? 1 : -1;`
- `x * y = 1 × 0 = 0`, 그리고 `0 <= 0`은 참이므로 `z = 1`.

$$\boxed{x = 1,\quad y = 0,\quad z = 1}$$
