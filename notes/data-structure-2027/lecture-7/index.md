# 자료구조 · 중간고사 요약 (W07)

중간고사 범위를 주제별로 정리한 요약 노트 모음입니다. 개념 설명과 함께, 핵심 자료구조는 브라우저에서 바로 돌려볼 수 있는 **C++ 실습(▶ 실행)** 으로 담았습니다. 재귀·복잡도 같은 기초부터 리스트·스택·큐, 우선순위 큐(힙), 트리·균형 트리까지 시험에 필요한 정의·규칙·성질을 빠짐없이 옮겼습니다.

| [리스트 · 스택 · 큐](./linear-structures) | 리스트, 스택, 큐, 원형 큐 |

::: tip 사용법
각 페이지의 C++ 코드 블록에는 **▶ 실행** 버튼이 있습니다. 버튼을 누르면 코드가 원격 컴파일 서버로 전송되어 **실제 `g++`** 로 컴파일·실행되고, 표준출력 결과가 아래에 그대로 표시됩니다. 코드를 직접 고쳐 가며 자료구조 연산이 어떻게 동작하는지 눈으로 확인해 보세요. 별도 설치 없이 브라우저만 있으면 됩니다.
