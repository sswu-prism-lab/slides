# 자료구조 · 강의노트

이 섹션은 슬라이드가 아닌 **노트·문제풀이형** 자료를 모읍니다. 자료구조 과목의 코드는 모두 **C++** 기반이며, 핵심 개념과 시험 문제에는 브라우저에서 바로 실행되는 **C++ 실습(▶ 실행)** 이 포함되어 있습니다.

::: tip 사용법
**요약 노트**(W07·W14)는 자료구조 주제별로 나뉘어 있고, 핵심 개념에는 `▶ 실행`으로 동작하는 **C++ 실습**이 들어 있습니다. 실습 상자는 브라우저에서 원격 컴파일 서버(wandbox.org)로 **실제 g++** 컴파일·실행 결과를 보여주며, `✎ 코드 수정`으로 직접 고쳐 다시 실행할 수 있습니다.

**시험 문제**(W08·W15) 페이지는 *문제 → 풀이(접힌 상태)* 순서이니, 먼저 스스로 풀어본 뒤 "풀이 및 해설"을 펼쳐 확인하세요. 순회·힙·그래프·해싱 등 알고리즘 문제는 **C++ 실습** 페이지에서 결과를 직접 재현할 수 있습니다.
:::
