# 문제 7 · 최소 신장 트리는 유일한가

::: info 문제 7 [2점]
임의의 그래프의 최소 신장 트리는 단일<span class="gloss">unique</span>한가? 그렇다면 이유를 서술하고, 그렇지 않다면 반례를 드시오.
:::

::: details 풀이 및 해설

**결론: 항상 유일하지는 않다.**

$$\boxed{\text{모든 간선 가중치가 서로 다르면 MST는 유일하지만, 같은 가중치가 있으면 유일하지 않을 수 있다.}}$$

**반례**

세 정점 $A,B,C$ 가 이루는 삼각형에서 세 간선의 가중치가 모두 **1**로 같다고 하자.

```text
      A
    1/ \1
    B───C
      1
```

신장 트리는 세 간선 중 **아무 두 개**를 고르면 되고, 세 경우 모두 총 가중치가 $1+1=2$ 로 최소이다.

- $\{A{-}B,\ B{-}C\}$
- $\{A{-}B,\ A{-}C\}$
- $\{B{-}C,\ A{-}C\}$

서로 다른 세 개의 MST가 존재하므로 **유일하지 않다**.

**유일성 조건**

모든 간선의 가중치가 **서로 다르면** MST는 반드시 유일하다. 절단 성질(cut property)에 따라 임의의 컷을 가로지르는 **가장 가벼운 간선은 반드시 MST에 포함**되는데, 가중치가 모두 다르면 "가장 가벼운 간선"이 매번 유일하게 정해지기 때문이다. 반대로 같은 가중치의 간선들이 하나의 사이클을 이루면, 위 반례처럼 어느 것을 넣어도 되어 여러 MST가 생긴다.
:::
