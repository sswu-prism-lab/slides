# 문제 6 · 노드 3개로 만드는 트리 개수

::: info 문제 6 [3점]
3개의 서로 다른 노드 $A$, $B$, $C$로 만들 수 있는 모든 트리<span class="gloss">tree</span> 구조의 개수가 몇 개인지 서술(1점)하고, 그 이유를 설명(2점)하시오.
:::

::: details 풀이 및 해설
**개수(1점).** 서로 다른 노드가 붙는 위치까지 구별하는 **뿌리 있는 트리(rooted tree)**로 세면 총 **9개**입니다.

$$\boxed{9\ \text{개}}$$

**이유(2점).** 두 가지 모양으로 나누어 셉니다.

**(1) 한 줄로 늘어선 사슬(경로) 모양** — 루트 아래 자식, 그 아래 손자.

- 루트가 될 노드를 고르는 경우의 수 3가지, 남은 두 노드 중 가운데(자식) 노드를 고르는 경우의 수 2가지 → $3 \times 2 = 6$개.

**(2) 루트가 두 자식을 갖는 모양** — 루트 하나 아래 나머지 두 노드가 형제.

- 루트가 될 노드를 고르는 경우의 수 3가지, 두 자식은 형제이므로 (순서를 구분하지 않으면) 1가지 → $3 \times 1 = 3$개.

두 경우를 합치면 $6 + 3 = 9$개입니다. 이는 라벨이 붙은 뿌리 트리의 개수 공식

$$n^{\,n-1} = 3^{\,3-1} = 3^2 = 9$$

와도 일치합니다($n=3$).

> 참고: 만약 형제 사이의 **좌·우 순서까지 다른 트리로 구분**한다면, (2)의 두 자식 배치가 2가지가 되어 $6 + 6 = 12$개가 됩니다. 보통 "트리 구조"는 형제 순서를 구분하지 않으므로 **9개**로 봅니다.
:::
