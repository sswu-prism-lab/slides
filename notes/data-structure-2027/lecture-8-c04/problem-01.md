# 문제 1 · 이진 힙의 원소 개수 범위

::: info 문제 1 [1점]
높이<span class="gloss">height</span>(또는 깊이<span class="gloss">depth</span>)가 $d$인 이진 힙<span class="gloss">binary heap</span>이 가질 수 있는 원소의 최댓값과 최솟값을 제시(각 0.5점)하시오.
:::

::: details 풀이 및 해설
이진 힙은 **완전 이진 트리(complete binary tree)**입니다. 높이가 $d$라는 것은 루트에서 가장 깊은 잎까지의 간선 수가 $d$, 즉 레벨이 $0$부터 $d$까지 있다는 뜻입니다.

- 레벨 $0$부터 $d-1$까지는 **꽉 차 있어야** 하므로, 그 노드 수는 $1+2+\cdots+2^{d-1} = 2^{d}-1$개입니다.
- 마지막 레벨 $d$에는 **최소 1개, 최대 $2^{d}$개**의 노드가 올 수 있습니다.

따라서

$$\text{최솟값} = (2^{d}-1) + 1 = 2^{d}, \qquad \text{최댓값} = (2^{d}-1) + 2^{d} = 2^{d+1}-1.$$

$$\boxed{\;2^{d} \;\le\; (\text{원소 수}) \;\le\; 2^{d+1}-1\;}$$

즉 최솟값은 $2^{d}$개(마지막 레벨에 잎 하나), 최댓값은 $2^{d+1}-1$개(포화 이진 트리)입니다.
:::
