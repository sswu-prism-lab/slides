# 문제 3 · $k$-평균 군집화

::: info 문제 3 — $k$-평균 군집화
1차원 상 데이터 포인트 집합 $\{1,2,4,5\}$가 주어졌다. 이를 $k=2$로 군집화하려고 할 때, 다음과 같은 초깃값에 대해 EM 알고리즘을 1회 적용한 결과를 보이시오.

$$
\mu_1=1,\qquad \mu_2=5
$$

---

E 단계에서는 초기 프로토타입 값에 대해 각각의 데이터 포인트를 할당한다. M 단계에서는 할당된 데이터 포인트를 정답으로 설정하고 프로토타입 값을 다시 구한다.
:::

::: details 풀이 및 해설
**E 단계 (할당)** — 각 점을 더 가까운 프로토타입에 배정한다.

| 점 | $\lvert x-\mu_1\rvert$ | $\lvert x-\mu_2\rvert$ | 배정 |
| :---: | :---: | :---: | :---: |
| $1$ | $0$ | $4$ | $\mathcal{C}_1$ |
| $2$ | $1$ | $3$ | $\mathcal{C}_1$ |
| $4$ | $3$ | $1$ | $\mathcal{C}_2$ |
| $5$ | $4$ | $0$ | $\mathcal{C}_2$ |

따라서 $\mathcal{C}_1=\{1,2\}$, $\mathcal{C}_2=\{4,5\}$.

**M 단계 (갱신)** — 각 군집의 평균으로 프로토타입을 다시 계산한다.

$$
\mu_1=\frac{1+2}{2}=\boxed{1.5},\qquad \mu_2=\frac{4+5}{2}=\boxed{4.5}
$$

1회 반복 후 프로토타입은 $(\mu_1,\mu_2)=(1.5,\,4.5)$로 갱신된다. (이 예는 한 번 만에 수렴하여, 다시 반복해도 할당과 프로토타입이 변하지 않는다.)

:::
