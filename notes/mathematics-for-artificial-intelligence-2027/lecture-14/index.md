# W14 · 기말고사 요약

기말고사 범위(9주차–13주차)의 핵심 개념을 한데 모은 **요약 정리 노트**입니다. 선형대수의 텐서 연산에서 출발하여 미분과 그래디언트, 행렬 미분, 그리고 이들이 합쳐져 하나의 신경망을 이루는 데이터 흐름까지, 시험 직전에 빠르게 훑을 수 있도록 정리했습니다. 각 페이지에는 개념을 손으로 확인해 볼 수 있는 실행 가능한 **Python 실습**을 함께 실었습니다.

> 제작: 성신여자대학교 AI융합학부 고원준 (<wjko@sungshin.ac.kr>)

## 목차

| 주차 | 주제 | 내용 |
| :---: | --- | --- |
| [선형대수](./linear-algebra) | 텐서와 행렬 | 스칼라·벡터·행렬·텐서, 크기·내적·사영·외적, 정방행렬·행렬식·역행렬·고윳값 |
| [미분](./differentiation) | 도함수와 그래디언트 | 도함수의 정의·미분법, 편미분, 그래디언트, 연쇄법칙 |
| [행렬 미분](./matrix-calculus) | 야코비와 헤시안 | 벡터·행렬 함수의 도함수, 야코비 행렬, 헤시안, 행렬 미분 항등식 |
| [신경망](./neural-networks) | 데이터 흐름 | 퍼셉트론, 합성곱(CNN), 순전파·역전파·경사하강 |

::: tip 읽는 법
용어는 **한국어**<span class="gloss">english</span> 형태로 원어를 함께 표기했습니다. 각 페이지의 실습 코드는 `▶ Run`으로 브라우저에서 바로 실행됩니다(최초 1회 런타임 로딩). 시험 범위는 선형대수(9–10주차) → 미분(11주차) → 행렬 미분(12주차) → 신경망(13주차)의 순서로 개념이 쌓여 있으니, 앞 페이지의 정의를 확실히 다진 뒤 다음으로 넘어가는 것을 권합니다.
:::
