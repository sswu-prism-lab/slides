# 자료구조실습 · 강의노트

이 섹션은 슬라이드가 아닌 **노트·문제풀이형** 자료를 모읍니다. 슬라이드 강의(2–6주차, 8–13주차)는 별도의 [슬라이드 사이트](https://sswu-prism-lab.github.io/slides/data-structures-and-applications-2027/)에서 제공됩니다.

자료구조실습 과목의 코드는 모두 **C++** 기반이며, 시험 문제 풀이에는 브라우저에서 바로 실행되는 **C++ 실습(▶ 실행)** 이 포함되어 있습니다.
